# Satellite Engineering Models
### International Competition of the Military Technical College Satellite Challenge (SATC-6)

This repository presents the engineering models developed by Team Satellites at the Arab Academy for Science, Technology and Maritime Transport (AASTMT) for the International Competition of the Military Technical College Satellite Challenge (SATC-6).

The project consists of three integrated satellite subsystems:

- Engineering Model Sun Sensor
- Smart Power System for Small Satellites
- Mission-Grade TT&C Communication System

Each subsystem was designed, implemented, integrated, and tested to demonstrate reliable operation and readiness for satellite applications.
